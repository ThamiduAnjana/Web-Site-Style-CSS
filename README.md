# Web-Site-Style-CSS @HTML @CSS
------------------

## Installation :

1. ***Download CSS***
------------------

2. ***Import/link CSS***

```
/*Import React*/

import './style.css';
```
```
/*HTML*/

<link rel="stylesheet" href="style.css">
```
------------------
